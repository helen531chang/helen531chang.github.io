    Story Synopsis

    While walking home from 7 Leaves one stormy day, Billy comes across a creepy, worn down house. Upon entering the house, Billy must choose between several different options which will determine life or death.



    Describe some programming concepts or structures you learned in this unit that you applied in your story. Think back to other stories you read, identify one or two stories you particularly like? Why?

    A programming structure that I learned in this unit that I applied in my story was the if statements. The if statements within the code allowed the user to choose between two different options throughout the story. I also learned about while loops and applied it to my story by allowing the user to type in either "A" or "B". If the user typed in a different letter, it would give them a message to try again. I personally enjoyed this one interactive story where my choice either led a guy to be safe or kidnapped. It was interesting!
